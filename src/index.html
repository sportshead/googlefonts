Usage

Prefix fonts.googleapis.com with %HOST%. For example:
  • https://fonts.googleapis.com/css2... -> https://%HOST%/fonts.googleapis.com/css2...
fonts.gstatic.com is also proxied and automatically prefixed by %HOST%: 
GET https://fonts.googleapis.com/css2?family=Roboto

@font-face {
  font-family: 'Roboto';
  font-style: normal;
  font-weight: 400;
  src: url(https://fonts.gstatic.com/s/roboto/v30/KFOmCnqEu92Fr1Me5Q.ttf) format('truetype');
}
===
GET https://%HOST%/fonts.googleapis.com/css2?family=Roboto

@font-face {
  font-family: 'Roboto';
  font-style: normal;
  font-weight: 400;
  src: url(https://%HOST%/fonts.gstatic.com/s/roboto/v30/KFOmCnqEu92Fr1Me5Q.ttf) format('truetype');
}
